# 02. デザイン方針（Design Policy）

## 方向性

**安心感・親しみやすさ・相談しやすさ。**
派手さより、地元密着のあたたかさ。「相談していい場所だ」と直感的に伝わることを最優先にします。

- 情報を詰め込みすぎず、余白でゆったり見せる
- 角丸・やわらかい影で、圧迫感を出さない
- 顔が見える／人の気配が伝わるトーン

## 色

唐津の海と松原をイメージした、落ち着いた地元カラー。
実際の値は [`src/styles/tokens.css`](../src/styles/tokens.css) で一元管理しています。

| 用途 | 変数 | 値 | 意図 |
|------|------|----|----|
| 背景 | `--color-bg` | `#fbf8f2` | あたたかいクリーム |
| メイン | `--color-primary` | `#2f7d5b` | 安心感のある深緑 |
| アクセント | `--color-accent` | `#e08a3c` | あたたかいオレンジ（CTA・強調） |
| LINE | `--color-line` | `#06c755` | LINEボタン |
| 本文 | `--color-text` | `#2f2a24` | 黒すぎない茶系ダーク |

> 色を調整するときは各コンポーネントではなく **必ず `tokens.css`** を編集してください。

## アイコン・イラスト

参考デザインに合わせ、**線画のSVGアイコン**で統一しています（絵文字は不使用）。

- アイコンは [`src/components/ui/Icon.tsx`](../src/components/ui/Icon.tsx) に集約。
  新規追加は `PATHS` に1件足すだけ。
- サービスごとに**色分け**（`data/services.ts` の `color`）。淡い丸背景（`.icon-box`）に乗せる。
- ヒーロー右の**人物イラストは「枠」だけ用意**（`.hero__art-slot`）。
  後からAI生成・いらすとや・撮影画像などを差し込む運用。
- 顔写真・実績写真も**プレースホルダ枠**（`.photo-slot`）を先に用意。届いたら差し替え。

## フォント

システムの日本語ゴシックを優先（外部フォント読込なし＝表示が速い）。
角ゴシック系で親しみと視認性を両立させます。詳細は `tokens.css` の `--font-base`。

## 余白

8pxグリッド基準（`--space-1`〜`--space-7`）。
セクション間はしっかり空け、読み手を急かさないリズムにします。

## ボタン

- 形は **ピル型（角丸最大）** で柔らかい印象
- 主要CTAはLINE緑またはアクセントのオレンジ
- ホバーで軽く浮く（`translateY`）程度の控えめな動き
- クラス：`.btn` + `.btn--line / --primary / --accent / --outline`

## スマホ表示の考え方

**モバイルファースト。** 唐津の地元利用者はスマホ閲覧が中心の想定。

- グリッドは `auto-fit` で自動的に1列へ折り返す
- スマホでは画面下部に **相談CTA（LINE・メール）を常時追従表示**（`StickyCta`）
- 文字サイズは `clamp()` で画面幅に追従
- タップ領域を十分に確保（ボタンは大きめ）

## 地元密着感・安心感を出す方針

- 誇張・煽りをしない（詳細は [`03_content_policy.md`](./03_content_policy.md)）
- 「できること・できないこと」を正直に見せる
- プロフィールで人柄を伝える
- 注意事項をきちんと載せることを、むしろ信頼につなげる
