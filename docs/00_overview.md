# 00. サイト概要（Overview）

このドキュメントは「唐津よろずサポート」LPの目的・方針の出発点です。
新しくこのプロジェクトに関わる人（人でもAIでも）は、まずここを読んでください。

---

## サイトの目的

地元の人が、頼み先に迷うような“ちょっと困った”を **気軽に相談できる窓口** をつくること。
一度きりのLPで終わらせず、将来的にサービス追加・料金表・実績・ブログなどを足して、
**地域密着のサポートサイトとして育てていく** ことを前提にしています。

## ターゲット

佐賀県唐津市周辺の地元の人。

- 個人
- 小規模事業者
- 高齢者
- 部活生
- 保護者

## コンセプト

> システム会社に頼むほどじゃない。便利屋さんに頼むほどでもない。
> そんな“ちょっと困った”を気軽に相談できる地元のサポート。

「何でも屋」ではなく **「困りごと相談パートナー」** という立ち位置。
安心感・親しみやすさ・相談しやすさを最優先にします。

## 主要サービス

1. **PC・AIサポート** … パソコン設定、スマホ相談、AI活用、Excel/データ整理、小さな自動化、HP・LP作成、EC/フリマ運営サポート
2. **暮らしのお手伝い** … 家具移動、荷物運び、草刈り、清掃、組立、軽作業
3. **不用品・資源整理サポート** … 不用品整理の相談、金属・スクラップの相談、売れるか/処分かの相談（※回収は行わない。法令配慮）
4. **バドミントンサポート** … 練習相手、基礎打ち、ノック、初心者レッスン、部活サポート（※学生時代に県大会優勝経験あり／現在は競技選手ではない）
5. **話し相手・付き添い** … 高齢者の話し相手、買い物/病院の付き添い、散歩・雑談、スマホ相談

サービスの詳細は [`04_service_list.md`](./04_service_list.md) を参照。
