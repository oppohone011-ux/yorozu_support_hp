# 04. サービス一覧（Service List）

サービスの正データは [`src/data/services.ts`](../src/data/services.ts) にあります。
このドキュメントは「対象者」「注意点」を含めた設計意図の整理です。
**文言を変えるときはコードとこの表の両方を更新** してください。

---

## 1. PC・AIサポート（slug: `pc-ai`）

「これどうやるの？」を、となりで一緒に。

- **内容**：パソコン設定 / スマホ相談 / AIの使い方 / Excel・データ整理 / 小さな自動化 / ホームページ・LP作成 / EC・フリマ運営サポート
- **対象者**：PC・スマホが苦手な人、個人・小規模事業者、これからAIを使いたい人
- **注意点**：初歩的な質問こそ歓迎。専門用語を使わず説明する。

## 2. 暮らしのお手伝い（slug: `kurashi`）

ちょっとした力仕事、まかせてください。

- **内容**：家具の移動 / 荷物運び / 草刈り / 清掃 / 組み立て / 軽作業
- **対象者**：一人で作業が難しい人、高齢者、業者に頼むほどでもない軽作業がある人
- **注意点**：内容・場所・状況によってはお受けできない場合がある。まず相談。

## 3. 搬入・現場・イベント設営（slug: `genba`）

現場仕事の経験、活かします。

- **内容**：店舗什器・家具の搬入 / 荷物運び・引越し補助 / 倉庫作業 / テント設営・撤去 / 会場設営・資材運搬 / 建設現場の補助（基礎・型枠まわり）
- **対象者**：店舗・イベント主催者、引越しや搬入で人手が欲しい人、現場の補助人員を探している人
- **注意点**：建設・測量・倉庫・イベント設営の実務経験あり。**専門工事そのものではなく補助・お手伝いが中心**と明記する（誇張しない）。

## 4. 不用品・資源整理サポート（slug: `seiri`）★法令配慮

「これ、どうしたらいい？」の整理から。

- **内容**：不用品整理の相談 / 金属・スクラップの相談 / 売れるか・処分かの相談
- **対象者**：物が溜まって困っている人、片づけの判断に迷っている人
- **注意点（最重要）**：
  - **「回収します」とは書かない・言わない。**
  - 回収業務は許可が必要なため **行わない**。
  - 行うのは片づけ・仕分け・売却/処分の判断などの **整理サポート**。
  - 必要に応じて **適切な業者を案内**。
  - 詳細は [`03_content_policy.md`](./03_content_policy.md) を必ず参照。

## 5. バドミントンサポート（slug: `badminton`）

楽しく上達したい人の、練習相手になります。

- **内容**：練習相手 / 基礎打ち / ノック練習 / 初心者レッスン / 部活サポート
- **対象者**：部活生、保護者、初心者、練習相手が欲しい人
- **注意点**：**学生時代に県大会優勝経験あり。現在は競技選手ではない。**
  誇張せず「基礎づくり・練習相手」として表現する。

## 6. 話し相手・付き添い（slug: `tsukisoi`）

そばにいてほしい時間も、サポートします。

- **内容**：高齢者の話し相手 / 買い物の付き添い / 病院の付き添い / 散歩・雑談 / スマホの相談
- **対象者**：高齢者、その家族、一人だと不安な場面がある人
- **注意点**：**医療行為・介護行為は行わない。** 日常のちょっとした付き添いが中心。

---

### サービスを追加するには

1. `src/data/services.ts` の `services` 配列に1件追加（`slug` は重複しないユニークな値）。
2. 必要なら `note` に注意書きを添える（法令・誇張への配慮）。
3. このドキュメントに対象者・注意点を追記。
4. 画面は自動でカードが増えます（`ServiceSection` がデータを回して描画）。
